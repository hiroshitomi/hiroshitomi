## 🔍 Proyecto actual – [Chequealo](https://github.com/hiroshitomi/chequealo)

**Chequealo** es una aplicación web que detecta cargos duplicados o sospechosos en resúmenes de tarjetas de crédito o débito.

- 🔎 Facilita el control financiero personal sin depender del ojo humano.
- 📄 Subís tu resumen y te ayuda a encontrar movimientos repetidos o con valores extraños.
- 💡 Pensada como una herramienta simple, útil y accesible para cualquier persona.

- Inscribite a la waitlist! (https://chequealo.vercel.app)
